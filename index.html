    极客时间-赵成的运维体系管理课

    课程地址：https://time.geekbang.org/column/intro/100003401
    课程名称：赵成的运维体系管理课
    主讲人：赵成，原蘑菇街平台技术总监，现在在七牛云（现在是2021年3月）
    课程内容：应用运维体系建设、效率和稳定性最佳实践、云计算时代的运维实践、个人成长
    我的评价：★★★★★
    学习耗时：一个周末（2021-03-06 & 2021-03-07）

    • 为啥学这门课：我目前是在一家创业公司（从第一行代码开始写的那种）做全栈开发，后端语言就是 Python，换成其他任何一种语言我都忙不过来o(╥﹏╥)o，前端的话会写 Vue、React + TypeScript，小公司比较自由，老板只要功能不管实现，当然前端主力还是另一个专职的前端小哥，然而后端 + 运维都是我。近一年以来我几乎所有的时间都在开发业务上的 crud 代码，单单做业务我就快累死了，运维以及部署相关的只能一切从简，无脑梭哈阿里云，简单一套 git + jenkins + k8s，代码仓库十几个，服务器近百台。最近去蘑菇街面试了一下运维发开（2021-03-04），我简历上没有完整的运维开发经验，全是在做业务，这居然有面试机会-.-，毫无意外的一面就挂了，面试我的小哥是个工作四年的运维开发（花名没听清，hhhhhh）。去面试的时候因为没有空闲的会议室，所以是在蘑菇街所处大楼一楼咖啡厅（蘑菇街自家开的）门口面的。面试整体下来比较轻松愉快，就是随便聊聊，主要聊到了大公司、微服务的 devops，怎么理解 devops，这个小哥建议我来看看他们前技术总监赵成在极客时间开的课。所以我就来学习一下，虽然我目前完全用不到（如果我先学完再去面试，可能蘑菇街面试就过了(～￣▽￣)～），但说不准以后有机会进啥大公司开始搞运维开发或者所谓的 SRE。暂时先无脑复制粘贴我觉得有必要的课程内容整理成 markdown，以后有机会再来精简吧。
    • 我从面试中了解到的蘑菇街信息：蘑菇街自研 CMDB，负责所有业务的发布构建部署，这个 CMDB 后端语言主要是 java；服务器数量 4000+，因为是腾讯系公司，所以用的腾讯云，最近刚开始进行容器化改造。
